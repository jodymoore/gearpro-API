GearPro: Login

API call:

gearpro.jodymoore.net/ap1/v1/login


Parameters:
username, password for related User_ID

Examples of API calls:

gearpro.jodymoore.net/ap1/v1/login

API response:
{
        "api_token": "955e465f6827c16bd1a1a9df4b1d26d2", 
}

Gearpro: update a pack Job

API call:

gearpro.jodymoore.net/ap1/v1/update


Parameters:
api-token, system_type, rig_ID

Examples of API calls:

gearpro.jodymoore.net/ap1/v1/update

API response:
{
        "firstName": "John",
        "System_Type": "Tandem",
        "rig_ID": "1" 
}
